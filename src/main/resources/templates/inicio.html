Web app Library

Permite crear, visualizar, editar y eliminar entidades como libros, autores y editoriales y almacenarlos en una base de datos relacional SQL.


Tecnologías usadas

Desarrollada en NetBeans, con Java 8 y Spring v2.6.5 , JPA, Html 5, CSS 3, Bootstrap 5, basado en el patrón de diseño modelo vista contraldor. (MVC). Posee 4 capas de nivel que se comunican en el siguiente orden: entidades, repositorios, servicios y controladores. Además posee la dependencia SpringSecurity que provee acceso mediante login.
